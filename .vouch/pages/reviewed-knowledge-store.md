Vouch maintains a curated knowledge store inside the repository — a set of durable, cited topic pages that capture agreed project context. [claim: vouch-starter-reviewed-knowledge] Rather than relying on session memory or ad-hoc notes, every load-bearing claim is reviewed before it enters the store and carries an inline citation back to its source. [claim: vouch-starter-reviewed-knowledge] This means a future agent or human starting a new session can read the wiki first and immediately recover the reasoning behind key decisions, without replaying conversation history. [claim: vouch-starter-reviewed-knowledge] The store is intentionally small and durable: ephemeral session records are raw material, not final pages. Only synthesized, topic-oriented content is promoted here.
